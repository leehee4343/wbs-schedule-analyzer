EmailJS 발송 설정
미설정 — 메일 앱으로 대체 발송
GitHub Pages 배포 시에는 저장소의 emailjs.config.js에 설정을 입력하면 모든 사용자가 실제 메일을 발송할 수 있습니다.
  1. emailjs.com 에서 무료 가입 (월 200건 무료)
  2. Email Services 탭 → Add New Service → Naver / Gmail 선택 후 SMTP 연결
  3. Email Templates 탭 → Create New Template, 아래 변수 사용:
    To: {{to_email}} · CC: {{cc_email}} · Subject: {{subject}} · Body: {{message}} · Reply-to: {{reply_to}}
    서식 포함 본문을 쓰려면 Body에 {{{message_html}}}를 사용하세요.
  4. Account → API Keys에서 Public Key 복사
  5. 배포용 설정 파일을 채우기 전에는 Public Key·Service ID·Template ID를 입력하고 이 브라우저에 저장을 누르세요. 발신 표시명과 회신 이메일은 선택 항목입니다.
